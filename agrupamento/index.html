Introdução

A linguagem HTML tem papel importante no desenvolvimento de sites, portais, e-commerce, aplicativos web e até APPs

Isso pois, o HTML permite criar páginas semanticamente formatadas trazendo o conteúdo organizado.

A linguagem possuí diversas TAGs de marcação, por exemplo as principais são:
<HTML>
Informa que o documento é um HTML.
<HEAD>
Trás o cabeçalho do arquivo HTML
<BODY>
Onde consta o conteúdo geral de uma página HTML
